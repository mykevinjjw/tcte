# 2026 統測倒數網頁 (2026 TCTE Countdown)

這是一個為 2026 年統一入學測驗（統測）設計的倒數計時網頁。

## 🔗 線上預覽
[https://mykevinjjw.github.io/tcte/](https://mykevinjjw.github.io/tcte/)

## 📁 專案結構
```
tcte/
├── assets/          # 靜態資源
│   ├── css/        # 樣式表
│   ├── js/         # 腳本
│   └── images/     # 圖片與圖示
├── index.html      # 主頁面
├── LICENSE         # 授權條款 (MIT)
└── README.md       # 專案說明
```

## 🚀 如何使用
1. Clone 此專案
2. 開啟 `index.html` 即可使用
