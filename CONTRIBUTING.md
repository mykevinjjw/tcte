# Contributing Guidelines / 貢獻指南

Thank you for your interest in contributing to the TCTE Countdown Webpage!
感謝您有興趣為統測倒數網頁做出貢獻！

## How to Contribute / 如何貢獻
1. Fork this project
   Fork 此專案
2. Create your feature branch (`git checkout -b feature/AmazingFeature`)
   建立您的功能分支
3. Commit your changes (`git commit -m 'Add some AmazingFeature'`)
   提交您的變更
4. Push to the branch (`git push origin feature/AmazingFeature`)
   推送到分支
5. Open a Pull Request
   建立 Pull Request

## Development Guidelines / 開發指南
- Please keep code clean
  請保持程式碼整潔
- Follow existing directory structure (assets/css, assets/js, etc.)
  請遵循現有的目錄結構
